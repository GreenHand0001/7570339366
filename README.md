# 项目简介

## 核心功能
- 基于YOLOv8的涡轮叶片表面缺陷检测
- 支持4类缺陷检测：['scratch','dot','crease','damage']
- 提供完整的源代码、300张标注数据集、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供实时检测界面，显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 可自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 提供项目运行教程和完整程序文件

## 适用场景
- 涡轮叶片表面缺陷的自动化检测
- 工业产品质量控制
- 制造业缺陷分析
- 技术研究与开发

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供UI界面，操作便捷
- 已配置好环境，可直接使用
- 包含完整的文件结构：程序文件、UI界面源文件、图标、数据集图片、运行教程
